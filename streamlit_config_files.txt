# FusionCRM - Google Sheets版

PicoCELA営業管理システムのGoogle Sheets対応版です。

## 🚀 セットアップ手順

### 1. Google Apps Script設定
1. [Google Apps Script](https://script.google.com/) にアクセス
2. 新しいプロジェクトを作成（名前: FusionCRM Backend）
3. 提供されたJavaScriptコードを貼り付け
4. 保存後、「デプロイ」→「新しいデプロイ」を選択
5. 種類: ウェブアプリ、実行者: 自分、アクセス: 全員
6. URLをコピーして保存

### 2. Streamlit実行
```bash
pip install -r requirements.txt
streamlit run fusion_crm_main.py
```

### 3. 接続設定
1. アプリケーション起動後、設定画面でGoogle Apps Script URLを入力
2. 接続テストを実行
3. 成功後、ダッシュボードが利用可能

## 📊 機能

- **リアルタイムデータ同期**: Google Sheetsと自動同期
- **企業管理**: 追加、編集、ステータス管理
- **ダッシュボード**: KPIメトリクスとビジュアライゼーション
- **ステータス履歴**: 変更履歴の自動記録
- **チーム共有**: Google Sheetsによる同時アクセス

## 🔧 トラブルシューティング

### 接続エラー
- Google Apps ScriptのURLが正しいか確認
- 「全員」にアクセス許可されているか確認
- ブラウザキャッシュをクリア

### データが表示されない
- Google Driveで「FusionCRM Database」スプレッドシートを確認
- 共有設定を確認
- Google Apps Scriptのログを確認
